Interactive Physical Computing

  • Caleb Cram
  • Game Controller Research

A series of projects building custom prototype hardware for user interactions with games.

CAPACITIVE THERAMIN:

A project in which I used an Arduino to build a Capacitive Theramin which uses ambient energy waves to power a speaker. This allows the user to "compose" music without touching the device.

TEMPSENSOR WITH LED INDICATORS:

A project in which I used an Arduino to sense the ambient temperature and light up a green, yellow or red LED based on the recorded temperature.

PROTOTYPE KEYBOARD:

This is a simple keyboard that I built using an Arduino.

DIRECTIONAL REMOTE CONTROL SYSTEM:

A project in which I used an Arduino to build a simple remote control system that could give N, S, E, W directions to a separate Arduino via LED's

PROTOTYPE GAME CONTROLLER

This is a custom game controller that was built using an Arduino connected to a joystick and an ultrasonic sensor.
